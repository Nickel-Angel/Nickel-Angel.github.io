乘法逆元

2021-02-12 学习笔记

0.前言

在 OIOIOI 中，大多数情况下，善良的出题人为了避免高精度等大整数计算，常常会要求输出答案对一个数（大多是质数）取模的情况，但这衍生了一个问题：若题目中计算需用到除法而我们知道，如果 a≡b(modc)a \equiv b \pmod{c}a≡b(modc) 在大部分情况下 ⌊ad⌋≡⌊bd⌋(modc)\lfloor \frac {a} {d} \rfloor \not\equiv \lfloor \frac {b} {d} \rfloor \pmod{c}⌊da​⌋​≡⌊db​⌋(modc)（注意一般题目会默认对一个数取模是数论（整数）意义上的取模，故这里除法为整数除法），这和等式的性质是不同的，要解决这个问题，就需要用到一个概念：乘法逆元。

本文主要介绍乘法逆元的定义以及其求法QwQ

1.定义

如果您到百度上搜索逆元，您会看到如下定义：（这里逆元素是逆元的全称）

逆元素是指一个可以取消另一给定元素运算的元素

---百度百科

这个定义似乎不太那么直观……

我们来举个例子吧，先再实数范围举例，由小学知识可知，如果一个代数式 FFF 乘一个数 aaa 后，再乘它的倒数 1a\frac {1} {a}a1​，相当于没有乘 aaa（这里不考虑 000 的情况），换句话说，我们乘 1a\frac {1} {a}a1​ 后，取消了代数式 FFF 乘 aaa 后值增大的影响。

不难发现这符合逆元的定义，故我们可以说一个数和其倒数互为乘法逆元。除此之外，我们还能发现一个数和其相反数互为加法逆元等等……

接下来回到代数式的例子，考虑为什么 aaa 的倒数 1a\frac 1 aa1​ 能消去乘 aaa 的影响。显然，是由于乘法结合律的存在，使得我们在运算 F×a×1aF \times a \times \frac 1 aF×a×a1​ 时可以先运算 a×1aa \times \frac 1 aa×a1​ 的值，再运算它和 FFF 的乘积，而 a×1a=1a \times \frac 1 a = 1a×a1​=1，任何数与 111 的乘积均为其本身，从而使乘 aaa 对 FFF 值增大的影响取消。

上文在实数范围内讨论了乘法逆元，现在我们来看本文主要讨论的内容，数论模意义下的乘法逆元。

由上文的分析来看，我们可以借助“任何数与111的乘积均为其本身”的性质定义模意义下的乘法逆元。

故其定义为：如果说 aaa 在模 ppp 意义下的乘法逆元是 xxx，那么 ax≡1(modp)ax \equiv 1 \pmod{p}ax≡1(modp)

我们通常将 aaa 的逆元记作 a−1a^{-1}a−1。

如同 000 没有倒数一样，000 也没有乘法逆元。

不难发现，aaa 在模 ppp 意义下的乘法逆元均为 [1,p−1][1,p - 1][1,p−1] 中的整数。

2.求逆元的方法

单个数求逆元模板

1.拓展欧几里得求逆元

不难发现 ax≡1(modp)ax \equiv 1 \pmod {p}ax≡1(modp) 可以写成 ax+py=1ax + py = 1ax+py=1 的形式：

将同余号换为等号得：

ax mod p=1 mod pax\,mod\,p = 1\,mod\,p axmodp=1modp

移项得：

(ax−1) mod p=0(ax - 1)\,mod\,p = 0 (ax−1)modp=0

由上式可知 ax−1ax - 1ax−1 为 ppp 的整数倍，不妨设其为 ppp 的 k(k∈Z)k(k \in Z)k(k∈Z) 倍：

ax−1=kpax - 1 = kp ax−1=kp

继续移项得：

ax−kp=1ax - kp = 1 ax−kp=1

令 y=−ky = -ky=−k，就写成了上文的形式：ax+py=1ax + py = 1ax+py=1

我们可以通过不断迭代的方式来求解单个数的逆元，这里涉及到了解线性同余方程的知识，可以看我的另一篇文章

我们发现这个方程有解的条件是 gcd(a,p)∣1gcd(a,p) \mid 1gcd(a,p)∣1，即 gcd(a,p)=1gcd(a,p) = 1gcd(a,p)=1，a,pa,pa,p 互质，所以得出结论：aaa 在模ppp 意义下的乘法逆元存在当且仅当 a,pa,pa,p 互质，这也在一定程度上解释了大多数题目模数为什么为质数的原因：（设模数为 ppp）可以保证在 [1,p−1][1,p - 1][1,p−1] 中的所有整数都有模 ppp 意义下的逆元。

而由于 gcd(a,p)=1gcd(a,p) = 1gcd(a,p)=1，故我们可以直接运用拓展欧几里得算法解 ax+py=gcd(a,p)ax + py = gcd(a,p)ax+py=gcd(a,p) 这个方程。

时间复杂度为 O(log⁡p)O(\log p)O(logp)。

模板题参考代码：

#include <cstdio>
#include <cstring>
#include <iostream>

using namespace std;

long long a, b;

int exgcd(long long a, long long b, long long &x, long long &y)
{
	if (b == 0)
	{
		x = 1, y = 0;
		return a;
	}
	int d = exgcd(b, a % b, y, x);
	y -= a / b * x;
	return d;
}

int main()
{
	scanf("%lld%lld", &a, &b);
	long long x = 0, y = 0;
	exgcd(a, b, x, y);
	printf("%lld", (x % b + b) % b);//这里防止出现负数
    return 0;
}

2.欧拉定理求逆元

观察式子 ax≡1(modp)ax \equiv 1 \pmod{p}ax≡1(modp)，我们发现它和欧拉定理 aφ(p)≡1(modp)a^{\varphi(p)} \equiv 1 \pmod{p}aφ(p)≡1(modp) 很像……

而欧拉定理又是 a,pa,pa,p 互质时成立，这和逆元存在的条件相同……

故将两个式子合在一起，得：

ax≡aφ(p)(modp)ax \equiv a^{\varphi(p)} \pmod{p} ax≡aφ(p)(modp)

两遍同除aaa得：

x≡aφ(p)−1(modp)x \equiv a^{\varphi(p) - 1} \pmod{p} x≡aφ(p)−1(modp)

故 x=aφ(p)−1 mod px = a^{\varphi(p) - 1}\,mod\,px=aφ(p)−1modp

我们可以在 O(p)O(\sqrt{p})O(p​) 的时间内求出单个数的欧拉函数。

引理：设 n=p1k1×p2k2×⋯×pmkm=∏i=1mpikin = p_1^{k_1} \times p_2^{k_2} \times \cdots \times p_m^{k_m} = \prod_{i = 1}^{m} p_i^{k_i}n=p1k1​​×p2k2​​×⋯×pmkm​​=∏i=1m​piki​​ 为正整数 nnn 的质数幂乘积表示式，则：

φ(n)=n×(1−1p1)×(1−1p2)×⋯×(1−1pm)=n×∏i=1m(1−1pi)\varphi(n) = n \times (1 - \frac {1} {p_1}) \times (1 - \frac {1} {p_2}) \times \cdots \times (1 - \frac {1} {p_m}) = n \times \prod_{i = 1}^{m} (1 - \frac {1} {p_i})φ(n)=n×(1−p1​1​)×(1−p2​1​)×⋯×(1−pm​1​)=n×∏i=1m​(1−pi​1​)

证明：

由于各质数幂之间是互质的，根据欧拉函数的性质可得：

φ(n)=∏i=1mφ(piki)=∏i=1mpiki×∏i=1m(1−1pi)=n×∏i=1m(1−1pi)\begin{aligned} \varphi(n) & = \prod_{i = 1}^{m} \varphi(p_i^{k_i}) \\ & = \prod_{i = 1}^{m} p_i^{k_i} \times \prod_{i = 1}^{m} (1 - \frac {1} {p_i}) \\ & = n \times \prod_{i = 1}^{m} (1 - \frac {1} {p_i}) \end{aligned} φ(n)​=i=1∏m​φ(piki​​)=i=1∏m​piki​​×i=1∏m​(1−pi​1​)=n×i=1∏m​(1−pi​1​)​

进而得证。

在求一个数的欧拉函数时，不妨将上式后面的分数部分通分化作 φ(n)=n×∏i=1m(pi−1pi)\varphi(n) = n \times \prod_{i = 1}^{m} (\frac {p_i - 1} {p_i})φ(n)=n×∏i=1m​(pi​pi​−1​)，我们可以在 O(n)O(\sqrt{n})O(n​) 的时间内枚举 nnn 的每个质因子，初始化 ansansans 为 nnn，每枚举到一个质因子 pip_ipi​ 就让 ans×pi−1pians \times \frac {p_i - 1} {p_i}ans×pi​pi​−1​，发现每个质因子只对答案贡献一次，故在统计完该质因子的答案后，这个质因子就无用了，为了方便最后判断我们是否将nnn的所有质因子全都枚举到，我们可以通过不断除该质因子来消去它，防止其再被计入答案。

求出欧拉函数后，可以使用快速幂求出答案。

故这样做的时间复杂度为 O(log⁡φ(p)+p)O(\log \varphi(p) + \sqrt{p})O(logφ(p)+p​)。

模板题参考代码：

#include <cstdio>
#include <iostream>
#include <algorithm>

using namespace std;

long long n, p;

long long phi(long long x)//求x的欧拉函数
{
    long long res = x, tmp = x;//初始化答案为x
    for (int i = 2; i * i <= tmp; ++i)
    {
        if (x % i == 0)
        {
            res = (res / i) % p * ((i - 1) % p) % p;//找到x的一个质因子,计算其对答案的贡献
            while (x % i == 0) x /= i;//统计完答案后,除去该质因子
        }
    }
    if (x > 1) res = (res / x) % p * ((x - 1) % p) % p;//防止漏筛质因子
    return res;
}

long long power(long long a, long long b)//快速幂
{
    long long res = 1;
    while (b)
    {
        if (b & 1) res = res * a % p;
        a = a * a % p;
        b >>= 1;
    }
    return res;
}

int main()
{
    scanf("%lld%lld", &n, &p);
    long long tmp = phi(p) - 1;
    printf("%lld", power(n, tmp));
    return 0;
}

3.费马小定理求逆元

注意，这种求逆元的方法仅使用在 ppp 为质数的时候

费马小定理：若 ppp 为质数，且整数 aaa 满足 p∤ap \nmid ap∤a，则有 ap−1≡1(modp)a^{p-1} \equiv 1 \pmod{p}ap−1≡1(modp)

对于定理中的式子，我们可以将两边同除aaa，得：

ap−2≡a−1(modp)a^{p - 2} \equiv a^{-1} \pmod{p} ap−2≡a−1(modp)

而 a−1a^{-1}a−1 就是 aaa 在模 ppp 意义下的乘法逆元。

故我们只需计算 ap−2 mod pa^{p - 2}\,mod\,pap−2modp 即可，而这个过程可用快速幂解决。

时间复杂度为 O(log⁡p)O(\log p)O(logp)

我们不妨看看本文开篇提出的问题，如何对一个分数取模？

一般的讲，我们是这么定义分数取模的：

我们设 ab mod p\frac {a} {b}\,mod\,pba​modp 的值为一个整数 xxx，其满足 bx≡a(modp)bx \equiv a \pmod{p}bx≡a(modp)且 x∈[0,p−1]x \in [0,p - 1]x∈[0,p−1]。

我们只需求出 xxx 的值即可。而上式两边同乘 bbb 在模 ppp 意义下的乘法逆元 b−1b^{-1}b−1 后可得：x≡ab−1(modp)x \equiv ab^{-1} \pmod{p}x≡ab−1(modp)

而 xxx 的范围在模 ppp 剩余系内，故可直接写作 x=ab−1 mod px = ab^{-1}\,mod\,px=ab−1modp

有理数取余模板题

在本题中，p=19260817p = 19260817p=19260817，是个质数，故可以直接使用费马小定理求逆元。

但注意：

1.1.1. 当 b≡0(modp)b \equiv 0 \pmod{p}b≡0(modp) 时，费马小定理不成立，故无解。

2.2.2. 本题 a,ba,ba,b 数据范围很大，long longlong\,longlonglong 也存不下，故需要写快读，边读边取模。

模板题参考代码：

#include <cstdio>
#include <cstring>
#include <cctype>
#include <algorithm>
#include <iostream>

using namespace std;

const int p = 19260817;

template<typename T>

{

		puts("Angry!");//b % p = 0 时无解

		printf("%d", (int)(1ll * a * power(b, p - 2) % p));

}

4.线性求逆

线性求逆模板题

注意，这种求逆元的方法仅使用在 ppp 为质数的时候

基于拓展欧几里得算法，我们虽然可以在 O(nlog⁡p)O(n \log p)O(nlogp) 时间内，求出 [1,n] (1≤n<p)[1,n]\,(1 \leq n < p)[1,n](1≤n<p) 中所有整数在模质数 ppp 下的乘法逆元，但在面对更大的数据范围时，我们就需要更快的方法。

首先，我们不难发现，对于任何正整数 ccc ，1−1≡1(modc)1^{-1} \equiv 1 \pmod{c}1−1≡1(modc)，即 111 在模任意正整数意义下的逆元为其本身。

然后设 p=ki+r (r<i,1<i<p)p = ki + r\,(r < i,1 < i < p)p=ki+r(r<i,1<i<p)，将其转化为同余式会得到：

ki+r≡0(modp)ki + r \equiv 0 \pmod{p} ki+r≡0(modp)

两边同时乘 i−1,r−1i^{-1},r^{-1}i−1,r−1，得：

i−1r−1(ki+r)≡0(modp)i^{-1}r^{-1}(ki + r) \equiv 0 \pmod{p} i−1r−1(ki+r)≡0(modp)

将式子乘开：

kr−1+i−1≡0(modp)kr^{-1} + i^{-1} \equiv 0 \pmod{p} kr−1+i−1≡0(modp)

移项得：

i−1≡−kr−1(modp)i^{-1} \equiv -kr^{-1} \pmod{p} i−1≡−kr−1(modp)

由 p=ki+r (r<i,1<i<p)p = ki + r\,(r < i,1 < i < p)p=ki+r(r<i,1<i<p) 易知 k=⌊pi⌋,r=p mod ik = \lfloor \frac {p} {i} \rfloor , r = p\,mod\,ik=⌊ip​⌋,r=pmodi，故有：

i−1≡−⌊pi⌋×(p mod i)−1(modp)i^{-1} \equiv - \lfloor \frac {p} {i} \rfloor \times (p\,mod\,i)^{-1} \pmod{p} i−1≡−⌊ip​⌋×(pmodi)−1(modp)

而 p mod i<ip\,mod\,i < ipmodi<i，故 p mod ip\,mod\,ipmodi 的逆元我们之前已经递推出来了，故我们可以通过这个式子推出 iii 的逆元是谁。

综上，我们在已知 111 的逆元为 111 的情况下，可以推出 [1,n] (1≤n<p)[1,n]\,(1 \leq n < p)[1,n](1≤n<p) 中所有整数在模质数 ppp 下的乘法逆元。

这个算法的时间复杂度为 O(n)O(n)O(n)。

注意 −⌊pi⌋- \lfloor \frac {p} {i} \rfloor−⌊ip​⌋ 为负数，但在模 ppp 意义下，其等价于 p−⌊pi⌋p - \lfloor \frac {p} {i} \rfloorp−⌊ip​⌋，故在实际应用中真正的递推式为：

i−1≡(p−⌊pi⌋)×(p mod i)−1(modp)i^{-1} \equiv (p - \lfloor \frac {p} {i} \rfloor) \times (p\,mod\,i)^{-1} \pmod{p} i−1≡(p−⌊ip​⌋)×(pmodi)−1(modp)

#include <cstdio>

}

5.离线逆元

离线逆元模板题

在 2019 年十二省联考中某个 std 长达 800~~（900？）~~ 多行的题中用到了这个新知识。

其实也不算全新的知识，似乎在以前这种方法只是被用来求阶乘逆元，最近才逐渐扩展到一般情况

（其中 pren−1pre_n^{-1}pren−1​ 已求出，pre0=1pre_0 = 1pre0​=1）

这种求逆元的方法的时间复杂度为 O(n+log⁡p)O(n + \log p)O(n+logp)，已经接近线性。

模板题参考代码：（注意常数优化和防爆 intintint）

#include <cstdio>
#include <cstring>
#include <cctype>
#include <iostream>
#include <algorithm>

using namespace std;

template<typename T>
inline void input(T &x)//本题卡常数,需要用快读
{
	x = 0;
	register char ch = getchar();
	register bool f = false;
	while (!isdigit(ch))
	{
		f |= (ch == '-');
		ch = getchar();
	}
	while (isdigit(ch))
	{
		x = (x << 1) + (x << 3) + (ch ^ 48);
		ch = getchar();
	}
	x = f ? -x : x;
}

const int maxn = 5e6 + 10;
int n, p, k, a[maxn], inv[maxn], pre[maxn], inv_pre[maxn];

inline int power(int a, int b, int p)
{
	int res = 1;
	while (b)
	{
		if (b & 1) res = 1ll * res * a % p;
		a = 1ll * a * a % p;
		b >>= 1;
	}
	return res;
}

int main()
{
	input(n), input(p), input(k);
	pre[0] = 1;//初始化
	for (register int i = 1; i <= n; ++i)
	{
		input(a[i]);
		pre[i] = 1ll * pre[i - 1] * a[i] % p;//计算前缀积
	}
	inv_pre[n] = power(pre[n], p - 2, p);//求出前缀积的逆元
	for (register int i = n; i > 0; --i)//递推求逆元
	{
		inv[i] = 1ll * pre[i - 1] * inv_pre[i] % p;
		inv_pre[i - 1] = 1ll * inv_pre[i] * a[i] % p;
	}
	int ans = 0;//计算题目中式子的值
	for (register int i = 1, t = k; i <= n; ++i)
	{
		ans = (1ll * inv[i] * t % p + ans) % p;
		t = 1ll * t * k % p;
	}
	printf("%d", ans);
	return 0;
}
